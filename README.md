## Galeria de dibujos

Una web simple donde se muestran varios dibujos digitales.

## Características

- 🔎 Al pasar el mouse sobre las imágenes se agrandan para poder observarlas mejor.

## Tecnología Utilizadas

- **HTML5:** Estructura y semántica de la página.
- **CSS3:** Diseño, maquetación.
- **JavaScript:** Funcionalidades e interactividad.
